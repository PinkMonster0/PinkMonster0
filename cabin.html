Cabin

Produced by: Zou Muyang

It's an individual task in NetEase employee training. The task aims to help us learn the pipeline of game art asset production.

I use Blender for modeling, Maya to fix UVs, Substance Painter and Substance designer to create the textures, and finally create the scene in Unreal Engine 4(Also, I add some weather effect like the rain and the fog).

Full game package available upon request.
